### File Structure

```
clonedperson
├── clonedperson.tar.gz
│   ├── train
│   │      ├── 1_c0001_2.jpg
│   │      ├── 236_c0064_1.jpg
│   │      ├── 1326_c0022_7.jpg
│   │      ├── 3107_c0039_1.jpg
│   │      ├── 4700_c0071_9.jpg
│   │      ├── ```
```



The filenames are encoded as follows. Take "3107_c0039_1.jpg" as an example,

- 3107 is the id of the person
- c0039 is the id of the camera
- 1 is the number of frames
